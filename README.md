# Kalas

Wedding & event planning app. React 19 + Vite + Tailwind CSS v4.

Design DNA: Cormorant Garamond (serif) + Hanken Grotesk (sans); cream `#F3F1E6`, forest `#3A4F37`, sage `#AEB080`, olive `#4A4E3C`.

## Development

```bash
npm install
npm run dev      # start dev server on http://localhost:5173
npm run build    # production build to dist/
npm run lint     # type-check
```
